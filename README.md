# Powerlets

⚡️ Chrome Extension to quickly find and run bookmarklets.

## Requirments

- NodeJS >=10.16
- Google Chrome

## Setup

```js
npm i
```

```js
npm run start
```

## Building

```js
npm run build
```

## Load extension in Google Chrome

- Go to `chrome://extensions/`
- Toggle on "Developer mode"
- Click "Load unpacked"
- Select the `/dist` folder after running the build script
